# موقع الهواتف الذكية

موقع بسيط لعرض أحدث الهواتف الذكية مع صور ومواصفات.  
يمكنك رفع هذا المشروع على GitHub Pages مباشرة.

## كيفية التشغيل

1. افتح `index.html` في المتصفح.
2. لإضافة هواتف جديدة، قم بتحرير مصفوفة `phones` في `script.js`.
3. كل هاتف يحتاج: `name`, `image`, `specs`.

## تقنية المشروع

- HTML
- CSS
- JavaScript
